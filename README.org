* Setup
** asdf

#+begin_src shell
# Setup the plugins repository
asdf plugin list all
# Install plugins
asdf plugin add java
# Check java versions
asdf list all java
# Install java version
asdf install java openjdk-20
# Set the global version for java
asdf global java openjdk-20
#+end_src
